# Weekly Engagement Review — Run Every Monday

Runs `skills/engagement-monitoring/SKILL.md` end-to-end across all three funnels. Runs **in addition to** (not instead of) `skills/athlete-brand-pipeline/SKILL.md`'s weekly roster pulse and `skills/cbh-advocacy/SKILL.md`'s weekly advocacy report — three distinct Monday reports in this repo, reflecting the three-funnel structure.

---

## The Monday Sequence (Run in This Order)

1. **Athlete-brand-pipeline weekly roster pulse** (per that skill) — covers the 36+ managed athletes specifically.
2. **CBH advocacy weekly report** (per that skill) — covers Margin's coach calls specifically.
3. **Engagement monitoring three-funnel pulse** (this skill) — covers public content performance across all three funnels, including but not limited to the managed-athlete content already covered in report #1.

Three reports, ~30-45 min total, all landing in `output/reports/` with the week number. Kyron's Monday read is roughly 15 minutes across all three — each report is built to be a fast, action-oriented read, not a data dump.
